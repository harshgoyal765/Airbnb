# Airbnb

---

## 🧰 Features

### Admin:
- 🔐 Admin login
- ➕ Add new homes
- ✏️ Edit/Delete homes

### User:
- 📝 Register / Login
- 🏡 View all available homes
- ❤️ Add homes to favorites
- 📅 Book a home
- 🗂️ View bookings and favorites

---

## 🔐 Authentication

JWT-based authentication for protected routes:

- Admin and User roles
- Middleware to protect routes accordingly
